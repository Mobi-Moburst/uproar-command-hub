# Uproar Command Center — Platform Plan

## Completed Features
- Coverage Intelligence (conversion funnels, reporter affinity, outlet momentum)
- Client report with outreach/conversion sections
- Moburst brand overhaul (light/dark mode, data viz)

## In Progress — Phase 2 Enhancements

### 1. Smart Reporter Matching (Building Now)
Given a client + topic/vertical, recommend the best reporters based on historical conversion data, placement history, and vertical affinity. Lives on the Intelligence page as a new tab.

### 2. Team Leaderboard (Next)
Gamified team rankings — monthly/quarterly stats, placement velocity, conversion rates, award wins. Strengthens the Teams section.

### 3. AI Coverage Summaries (After)
Auto-generate natural-language monthly performance narratives per client using Lovable AI. Edge function + UI integration.
